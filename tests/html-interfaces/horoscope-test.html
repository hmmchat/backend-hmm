⭐ Horoscope Test Interface

Test Horoscope API - User Service

Base URL: http://localhost:3002
Service: User Service (port 3002)
Note: Images may not display if not uploaded to files-service yet

🔮 Get User Horoscope

Enter a user ID to get their horoscope based on their date of birth. The horoscope is calculated from the mandatory dateOfBirth field.

Image URL:

🧪 Available Test Endpoints

Public Endpoint: GET /users/:userId/horoscope
Test Endpoint (No Auth): GET /users/test/:userId/horoscope
Authenticated Endpoint: GET /me/horoscope (requires Bearer token)

📋 API Response Format

{
  "horoscope": {
    "name": "Aries",
    "imageUrl": "https://cdn.hmmchat.live/horoscopes/aries.png"
  }
}

ℹ️ How Other Services Can Use This

Other services (like discovery-service) can call this API to get horoscope for users: 

// From discovery-service or any other service
const response = await fetch('http://user-service:3002/users/{userId}/horoscope');
const { horoscope } = await response.json();
// Returns: { horoscope: { name: "Aries", imageUrl: "..." } }
