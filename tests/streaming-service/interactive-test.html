🎯 Dares with Gifts

💡 Quick Test Flow:

  1. Create room with 2+ users (matched/squad call)
  2. Connect all users via WebSocket
  3. Have users join the room
  4. View Dare: User1 views dare-1 (others see it sync)
  5. Assign Dare: User1 assigns dare-1 to User2 (required for 3+ users, optional for 2 users)
  6. Get Gifts: Load gift list
  7. Send Dare: User1 sends dare-1 with gift (100% payment transferred immediately, auto-assigns for 2 users)

🎮 Interactive Playground

This is a playground for testing the streaming service. All features are available above:

  • Room Management: Create rooms (matched/squad calls), join, leave
  • Host Features: Kick participants, start/stop broadcasting (HOSTs only)
  • Chat: Send messages, view chat history
  • Broadcasting: Start/stop broadcast, join as viewer
  • Dares: View dares (real-time sync), assign to users, send with gifts, 100% payment on send
  • Icebreakers: Get random conversation starters (displayed only to you, no rewards)
  • Room Info: View room details including HOST/PARTICIPANT roles

💡 Tip: Create a matched call (2 users = both HOSTS) or squad call (all users = HOSTS), then explore!
